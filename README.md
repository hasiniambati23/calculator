# 🧮 Calculator Web Application

A responsive and user-friendly calculator built using HTML, CSS, and JavaScript. This project performs basic arithmetic operations with a clean interface and interactive user experience.

## ✨ Features
- Basic arithmetic operations

- Addition (+)

- Subtraction (-)

- Multiplication (×)

- Division (÷)

- Clear/Delete functionality

- Responsive design

- Real-time input display

- Interactive button effects

- Error handling for invalid operations



## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
